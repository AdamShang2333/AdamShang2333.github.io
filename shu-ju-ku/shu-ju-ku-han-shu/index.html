# 操作函数

# 1. 字符串操作函数

# as 关键字

给列或者表起别名，将过长的表名简化
特别注意:

1. 只在当前 SQL 语句内生效，到下一个 SQL 语句就失效 ---> 不是全局 "改名"
2. 在名字后直接加空格，不加 as 关键字也行

select concat('张三', '李四', 'ddd') as name;

as 也可以不加 直接在名字后面加 ' '(空格) 后接关键字

// age sage :等价于 age as sage
select s.name as sname, age sage, gender
from student as s;

# insert 替换

格式: insert (stringA,m,n,stringB)
将 A 从第 m 个字符开始，替换 n 个；这 n 个替换为 B .
序号从 1 开始

select insert('这是一个字符串', 5, 3, '喵喵喵');
// 输出结果: 这是一个喵喵喵

# upper/lower 大小写转换

类比 string 的方法

select upper('aaa'); // 结果:AAA
select lower('HHH'); // 结果: hhh

# substring 截取子串

select substring('这是一个字符串', 2);
// --> 等价于:
	select substring('这是一个字符串' from 2);

select substring('这是一个字符串', 2, 3);

# 2. 时间操作函数

# curtime () 获取当前时间

# curdate () 获取当前日期

# now () 获取当前时间和日期

# mysql 的时间类型 :

1.date ：只保存日期
2.timestamp ：时间戳 能保存时间和日期，根据时区进行修正 (时区不一样 看到的时间也不同)
　　　　　　(范围: 1970~2038 年 最大到 2038 年 1 月 19 日 3:14)
3.datetime 能保存时间和日期，不会根据时区进行修正；(最大到 9999-12-31 23:59:59.9999999)

select year(now());
select month(now());
# 一年中的第几周
select week(now());
select hour(now());

# 两个日期间相隔的天数 (昨天,今天 相隔1天)
select datediff(now(), '2019-1-1');
# 计算xx天之后的日期
select adddate(now(), 20);

# 生成一个随机数
select rand();

select floor(7.9);
# 向上取整 --> 8

# 聚合函数

对数据进行集中操作的函数 ---> 对整列数据进行处理

注意:
1. 多个聚合函数可以一起写
2. 聚合函数的返回不能和列名一起 会报错
　(返回的都是一列的值 不能和某个条件查询的某行值一起返回)
３. 如果使用了 group by 的列，可以和聚合函数写在一起

# 求数量 count (*)

注意:　填 * 不会过滤空行；填分组名字会过滤掉空行 (空值)
---> 计算数量最好使用 保留空行 *

# 2. 查询每个学员的成绩在90以上的课程有多少门(成绩90以上, 每个学员有几门课程)
# 填 * 不会过滤空行 填分组名字会过滤掉空行(空值)
# 计算数量最好使用* 保留空行
select count(*), sname
from Score
where grade >= 90
group by sname;

# having 子句

对分组的结果进行过滤 (where 对每一个结果进行过滤)
　１.having 的条件一般都是对聚合函数的判断！！
　２.where 的条件一般都是对列的判断.

select avg(grade), sname
from Score
group by sname
having sum(grade) > 300;

# on , where 和 having 的使用:

!!!on 最先执行，where 次之 having 最后执行

 on:使用join联表的时候使用
 如果条件是针对表中的行,就用where  (对本身的各条数据有限制)
 如果条件是针对分组的,就用having

# group by 子句

多见"每个,各个"这种词 使用分组

该子句后面跟某一列的列名 表示按该列进行分组
--> 分组列中值一样的数据 放入一个 "小组" , 可以对其使用聚合函数

1. 当一列被标记为分组，那么该列可以写在查询结果中
2. 只要进行分组，基本都和聚合函数有关系
3. 可以有多层分组，以 "," 分隔接在 group by 后面

# distinct 关键字

查询结果不重复

select distinct depart
from Employee;
