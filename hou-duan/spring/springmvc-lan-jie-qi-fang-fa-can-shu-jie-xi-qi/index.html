# SpringMVC 拦截器 & 方法参数解析器

# 拦截器

拦截器是 SpringMVC 中的技术
　　　针对 Handler (处理器) 进行的拦截；
过滤器是 Serlvet 中的技术
　　　针对 Request (请求) 进行拦截

常见使用:实现HandlerInterceptor接口,重写preHandle 判断请求是否可以进入Handler

# 方法参数解析器

方法参数解析器的作用
就是在 Controller 中，能在接口方法传入的参数中设置任意的类
并且通过框架能直接在方法中调用传入的参数 (类).
对传入参数映射到对应类型的过程 就需要方法参数解析器

Step1 自定义一个注解,标识执行自定义方法参数解析器的实体类

Step2 自定义方法参数解析的接口实现类

Step3 在接口中,使用自定义注解标识使用自定义方法参数解析器的类
