# 反射

# 反射机制的概念

将类的各个组成部分，封装为其他对象

在运行状态中，
对任意一个类，都能够知道这个类的所有属性和方法；
对任意一个对象，都能够调用它的任意一个属性和方法；

# 反射的功能

1 . 在程序的运行过程中，可以操作这些对象

  1. 可以解耦，提高程序的可扩展性
