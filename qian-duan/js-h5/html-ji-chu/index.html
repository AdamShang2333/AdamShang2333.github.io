# html 模板

<!--申明一个HTML文档-->
<!DOCTYPE html>
<!--代表一个网页的整体-->
<html>
	<!--展示网页标题，样式等-->
	<head>
		<!--设置编码格式，防止乱码（UTF-8 常用编码格式）-->
		<meta charset="UTF-8">
		<!--设置网页标题-->
		<title>百度一下</title>
	</head>
	<!--网页主体（直观见到的）-->
	<body>
		
	</body>
</html>

# 标签分类

  • 模板 功能类

     html body head link title style meta strong  em   
 link:

  • 块标签

     div h1~h6 p ul ol li  br hr

    特性:

    1. display:block
    2. 占父级的一行
    3. 可以嵌套块标签和行标签 (p 标签只能嵌套行标签)
    4. 居中：给本身加 margin: 0 auto
    5. 可以设置宽高，不设置高度时，由内容撑开

  • 行标签

span a img i  b sub sup input

特性

  1. display:inline
  2. 多个行标签可在一行显示
  3. 不能设置宽高，默认由内容撑开
  4. 行标签可以嵌套行标签
  5. 居中：给父标签加 : text-align:center
  6. 垂直方向 margin,padding 失效
  • 表单
form tbody  label input select textarea
  • 表格
table tr td th

### 浮动

浮动最早是解决文字环绕问题，现在一般用来做左右布局使用，使用时注意清浮动.

清浮动: overflow : hidden

浮动元素的特性

  1. 浮动元素不占文档流
  2. 设置成浮动后不再具有行块标签的特性，称为浮动元素
  3. 浮动元素可以设置宽高，默认由内容撑开

注意：浮动元素的兄弟标签都要为浮动

# 定位

  • static 默认值

  • fixed 固定定位

    特性:

    1. 相对于浏览器定位
    2. 不占文档流
    3. 不再是行块标签，成为定位元素
    4. 可以设置宽高，由内容撑开
    5. 一般用作导航 / 导航弹出框等

  • relative 相对定位

    特性:

    1. 相对于自身定位
    2. 占文档流
    3. 不改变标签本身特性
    4. 一般用来做微调或者是绝对定位的父级

  • absolute 绝对定位

特性:

  1. 相对于最近的定位父级 (从父标签往外找) , 如果没有定位父级，相对于浏览器定位
    2. 不占文档流
    3. 不再是行块标签，成为定位元素
**top / left / right / bottom / z-index :只能和定位配合使用**  
**定位居中**

# 响应式布局

更适用于移动端，来对不同的屏幕做适配，PC 端使用的话，页面不会太复杂

移动端布局，必须添加 viewport

# 流式布局 / 百分比布局

  1. 宽度：本身宽度 / 父级的宽度

  2. 高度：本身高度 / 父级的高度

  3. 定位值 :

    纵向定位 (top bottom) / 定位父级的高度
    横向定位 (left right) / 定位父级的宽度

  4. 盒模型 (margin padding)

    盒模型纵向 纵向值 / 父级的宽度盒模型横向
    横向值 / 父级的宽度？: 多数情况高度由文档流撑开
