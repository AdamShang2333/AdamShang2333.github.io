## 新增标签

### 音频标签 - audio

可以设置宽、高
支持文件格式：mp3 wav ogg
audio 中可以放置文本，在浏览器不支持此标签时显示。可以插入背景图

<audio " src="source/群星 - 大力哥 浑身难受 Rap.mp3"   
   autoplay="autoplay"   
   controls="controls" 
   loop="loop"
   preload="auto">

### 视频标签 video

 autoplay="autoplay"  
  muted="muted" //静音   
  width="400"   
  height="600"   
  controls=""   
  poster="添加图片">  
  </video>
		
		<video width="800" height="">
			自动匹配可用的文件格式
			<source src="myvideo.mp4" type="video/mp4"></source>
			<source src="myvideo.ogv" type="video/ogg"></source>
			<source src="myvideo.webm" type="video/webm"></source>	
			<object width="" height="" type="application/x-shockwave-flash" data="myvideo.swf">
				<param name="movie" value="myvideo.swf" />
				<param name="flashvars" value="autostart=true&amp;file=myvideo.swf" />
			</object>
			当前浏览器不支持 video直接播放，点击这里下载视频： <a href="myvideo.webm">下载视频</a>
		</video>

## 新增选择器
### 属性选择器
#### 语法例: 标签后 [] 中接属性，选择包含特定属性的元素
#### 无优先级，看执行顺序

<style type="text/css">
   a[href]{
   		}     []内  
   		p[class]{
				color: deepskyblue;
			}
			
	a[href="####"]{
				color: gray;
			}
</style>
	<body>
		<!--a[href="#"]{}-->
		<a href="#" class="title">超链接</a>
		<p class="title">中午吃点啥</p>
		<p>我也不知道</p>
		<a href="#">超链接</a>
		<a href="####">超链接</a>
		<a href="#">超链接</a>
		<a href="#">超链接</a>
		<a >cc</a>
	</body>

### 结构选择器
1. 子代元素选择
div>span
只能选择儿子辈标签，不能跨级选择

2. 紧邻元素选择
span+a
对紧接着的 span 下面的第一个 a 标签进行选择，其余的同类标签不会选中

### 伪类选择器
:not "反向" 选择
:not (.color)

/*未设置成.color的*/
			:not(.color){
				color: blue;
			}
			
			:not(div .color){
				color: goldenrod;
			}

:nth- "序列" 选择
可以设置 n 为负值，eg:nth-of-type (-n+5)--n
从 0 开始取值，只能选中前 5 个子代元素

 /*所有父标签中的第二个标签,如果是p标签,选择器可以选中*/
			/*第二个不是p标签则无效*/
			p:nth-child(2){
				color: red;
			}
			<!--倒序查找选择-->
			p:nth-last-child(4){
				color: red;
			}
			p:nth-of-type(2){
				color: blue;
			}
			 
			/*n从0开始*/
			p:nth-of-type(2n+1){
				color: gray;
			}
			
			/*倒数第几个*/
			p:nth-last-of-type(1){
				color: deepskyblue;
			}

			p:first-of-type{
				color: red;
			}
			p:last-of-type{
				color: greenyellow;
			}
			
			/*所有父标签中p标签只有一个*/
			p:only-of-type{
				color: red; 
			}
			
			/*所有父标签中只有一个子标签,并且是p标签*/
			p:only-child{
				color: skyblue;
			}
