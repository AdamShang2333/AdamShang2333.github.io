##transition (过渡样式)
### 可设定过渡的时间 延时时间 速度曲线 指定某个过渡的元素
#### 过渡时间 transition-duration: 1s;
#### 过渡延时 transition-delay: 1s;
#### 过渡效果 transition-timing-function: cubic-bezier (0, 0, 0, 1.97) ;
常用的预设过渡效果:ease ease-in ease-inout ease-out
### 过渡常用默认样式:transition:all 1s

##transform (变换)
### 二维效果
transform: translate / 平移 /

移动参照系为自身 使用 - 50% 可做居中效果
可分别设置 x y 轴的平移距离 亦可写成复合 (20px,20px)
平移数值可以为负值

transform: rotate / 旋转 /

不设置时默认为以 z 轴旋转
可分别设置 x y 轴旋转
旋转数值可以为负值

transform: scale / 缩放 /

默认以缩放标签的中心做缩放基点
可分别设置 x y 轴缩放，亦可写成复合 (2,2)
缩放数值可以为负值

transform: skew / 倾斜 /

可分别设置 x y 轴倾斜，亦可写成复合 (30deg,30deg)
倾斜数值可以为负值

#### 二维变换组合

transform 可以将多个变换结合使用，但是要注意变换顺序！！

div:hover{
	background-color: red;
	/*执行顺序会对变换效果造成影响 即先转 再平移 放大;平移 旋转 放大*/
	transform: rotate(360deg) scale(2) translateX(100px) ;
	}

#### 变换基准点自定义
transform-origin: left top;

基准点可设置特定位置或者具体数值
x 轴:left center right
y 轴:top center bottom

### 三维效果

三维效果需要在父级标签设置 "3d 盒子", 景深
使用时，与二维基本一致。可设置负方向数值

		.all{
				width: 500px;
				height: 500px;
				border: 1px solid black;
				margin: 100px auto;
				/*在父级设置3D盒子*/
				transform-style: preserve-3d ;
				
				/*景深 */
				perspective: 1000px;
